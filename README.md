# Kurban Bayramı tatil haritası

Mayıs 2026 rotası: Leaflet + OpenStreetMap ile etkileşimli harita ve günlük özetler.

## Site (GitHub Pages)

https://tunasen.github.io/tatil-plani/

**404 görürsen:** Repoda henüz Pages açılmamıştır — push tek başına siteyi yayınlamaz.

1. GitHub’da **TunaSen/tatil-plani** → **Settings** → sol menüden **Pages**
2. **Build and deployment** → **Source:** **Deploy from a branch**
3. **Branch:** `main`, klasör **/ (root)** → **Save**
4. 1–3 dakika sonra sayfayı yenile (ilk deploy bazen biraz sürer)

Ücretsiz hesapta repo **public** olmalı; özel repo için GitHub’un ücretli Pages kuralına bakın.

## Yerel açılış

`index.html` dosyasını tarayıcıda açın (CDN için internet gerekir).
